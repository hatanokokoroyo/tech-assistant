# 前端目录结构

基于 Vue 3 + Vite + TypeScript + Element Plus。

```
frontend/
├── public/                          # 静态资源（直接复制，不经过构建）
│   └── favicon.ico
│
├── src/
│   ├── api/                         # API 调用层
│   │   ├── request.ts               # Axios 实例 + 拦截器（JWT 注入）
│   │   ├── auth.ts                  # POST /api/auth/*
│   │   ├── projects.ts              # GET/POST/PUT/DELETE /api/projects/*
│   │   ├── repos.ts                 # GET/POST/DELETE /api/projects/*/repos/*
│   │   ├── ssh-keys.ts              # GET/POST/DELETE /api/ssh-keys/*
│   │   ├── files.ts                 # GET/PUT /api/projects/*/files/*
│   │   ├── conversations.ts         # GET/POST/DELETE /api/projects/*/conversations, /api/conversations/*
│   │   ├── stream.ts                # POST /api/conversations/*/stream（SSE）
│   │   ├── events.ts                # GET/POST /api/projects/*/events
│   │   └── admin.ts                 # GET /api/admin/*
│   │
│   ├── assets/                      # 样式、图片等静态资源
│   │   ├── styles/
│   │   │   ├── variables.scss       # 全局样式变量
│   │   │   └── global.scss          # 全局样式
│   │   └── logo.svg
│   │
│   ├── components/                  # 通用可复用组件
│   │   ├── MarkdownEditor.vue       # Markdown 编辑器封装
│   │   ├── BranchSwitcher.vue       # 分支切换组件
│   │   └── SshKeyUploader.vue       # SSH 密钥上传组件
│   │
│   ├── layouts/                     # 页面布局
│   │   ├── AppLayout.vue            # 主布局（三栏结构）
│   │   └── AuthLayout.vue           # 登录/注册页布局（无侧栏）
│   │
│   ├── router/
│   │   └── index.ts                 # Vue Router 配置 + 路由守卫
│   │
│   ├── stores/                      # Pinia 状态管理
│   │   ├── auth.ts                  # 用户认证状态
│   │   └── chat.ts                  # 对话状态（当前会话、消息列表）
│   │
│   ├── views/
│   │   ├── Login.vue                # 登录页
│   │   ├── Register.vue             # 注册页
│   │   ├── Dashboard.vue            # 首页概览（默认跳转到项目列表）
│   │   ├── project/
│   │   │   ├── ProjectList.vue      # 定制项目列表（最左侧导航栏）
│   │   │   └── ProjectDetail.vue    # 定制项目详情（中间导航栏 + 右侧内容区）
│   │   │       ├── FilePanel.vue    # 文件列表面板（中间）
│   │   │       └── DocEditor.vue    # 文档编辑器（右侧）
│   │   └── chat/
│   │       ├── ChatPanel.vue        # 对话列表面板（中间）
│   │       └── ChatView.vue         # AI 对话界面（右侧，流式消息列表）
│   │           └── components/
│   │               ├── MessageList.vue       # 消息列表（流式渲染）
│   │               ├── MessageItem.vue       # 单条消息
│   │               ├── ReasoningBlock.vue    # 思维链（默认折叠）
│   │               ├── ToolCallBlock.vue     # 工具调用（默认折叠缩略）
│   │               └── ChatInput.vue         # 输入框 + 发送
│   │
│   ├── utils/
│   │   ├── sse.ts                   # SSE 客户端封装
│   │   └── format.ts               # 时间、文本格式化
│   │
│   ├── App.vue                      # 根组件
│   └── main.ts                      # 入口文件
│
├── Dockerfile                       # 多阶段构建（build → nginx）
├── nginx.conf                       # Nginx 配置（反向代理 /api → backend）
├── package.json
├── tsconfig.json
├── vite.config.ts
└── .env                             # VITE_API_BASE_URL
```

## 路由设计

| 路径 | 视图 | 说明 |
|---|---|---|
| `/login` | Login.vue | 登录 |
| `/register` | Register.vue | 注册 |
| `/` | ProjectList.vue | 项目列表（重定向到第一个项目或空状态引导） |
| `/projects/:id` | ProjectDetail.vue | 项目详情，包含文件和对话两个子面板 |
| `/projects/:id/files` | ProjectDetail → FilePanel + DocEditor | 文件面板 |
| `/projects/:id/chat` | ProjectDetail → ChatPanel + ChatView | 对话面板 |
| `/projects/:id/chat/:conversationId` | ProjectDetail → ChatPanel + ChatView | 指定对话 |
