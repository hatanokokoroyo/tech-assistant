# 前端目录结构

基于 React 19 + Vite + TypeScript + Tailwind CSS v4 + shadcn/ui。

```
frontend/
├── index.html                       # SPA 入口（lang="zh-CN"）
├── package.json                     # 依赖与脚本
├── tsconfig.json                    # TypeScript 配置
├── vite.config.ts                   # Vite + React SWC + Tailwind + 路径别名
├── components.json                  # shadcn/ui 配置
│
├── Dockerfile                       # 多阶段构建（node:20-alpine → nginx:stable-alpine）
├── nginx.conf                       # Nginx 配置（反向代理 /api → backend）
│
└── src/
    ├── main.tsx                     # 入口：挂载 React root
    ├── App.tsx                      # 根组件：QueryClientProvider + RouterProvider + Toaster
    ├── index.css                    # Tailwind 入口 + @theme 全局设计 token
    ├── vite-env.d.ts                # Vite 类型声明
    │
    ├── lib/                         # 核心工具
    │   ├── utils.ts                 # cn()（clsx + tailwind-merge）
    │   ├── api-client.ts            # fetch wrapper（JWT 注入 + 401 拦截 + 错误规范化）
    │   └── format.ts                # 时间格式化（formatDistanceToNow）
    │
    ├── components/
    │   ├── ui/                      # shadcn/ui 组件（Radix UI + Tailwind）
    │   │   ├── button.tsx           # 按钮（6 种变体 + 4 种尺寸）
    │   │   ├── input.tsx            # 文本输入框
    │   │   ├── textarea.tsx         # 多行输入
    │   │   ├── label.tsx            # 表单标签
    │   │   ├── dialog.tsx           # 模态对话框
    │   │   ├── alert-dialog.tsx     # 危险操作确认弹窗
    │   │   ├── dropdown-menu.tsx    # 下拉菜单
    │   │   ├── tabs.tsx             # Tab 切换
    │   │   ├── scroll-area.tsx      # 自定义滚动条
    │   │   ├── tooltip.tsx          # 悬停提示
    │   │   ├── badge.tsx            # 状态标签
    │   │   ├── collapsible.tsx      # 折叠面板
    │   │   ├── separator.tsx        # 分割线
    │   │   ├── skeleton.tsx         # 骨架屏
    │   │   └── avatar.tsx           # 头像
    │   │
    │   └── shared/                  # 项目级共享组件（预留）
    │
    ├── hooks/                       # 自定义 Hooks
    │   ├── use-sse.ts               # SSE 流式连接管理（fetch + ReadableStream）
    │   └── use-auto-scroll.ts       # 消息列表自动滚动
    │
    ├── stores/                      # Zustand 客户端状态
    │   ├── auth-store.ts            # JWT token + 用户信息（persist 到 localStorage）
    │   └── app-store.ts             # 选中项目 ID 等 UI 状态
    │
    ├── api/                         # API 调用函数（纯函数，无副作用）
    │   ├── auth.ts                  # POST /api/auth/*
    │   ├── projects.ts              # CRUD /api/projects/*
    │   ├── conversations.ts         # CRUD /api/conversations/*
    │   ├── files.ts                 # GET/PUT /api/projects/*/files/*
    │   ├── repos.ts                 # CRUD /api/projects/*/repos/*
    │   └── ssh-keys.ts              # CRUD /api/ssh-keys/*
    │
    ├── queries/                     # TanStack Query hooks（缓存/刷新/乐观更新）
    │   ├── keys.ts                  # 统一 Query Key 前缀
    │   ├── use-projects.ts          # useProjects / useCreateProject / useUpdateProject / useDeleteProject
    │   ├── use-conversations.ts     # useConversations / useMessages / useCreateConversation / useDeleteConversation
    │   ├── use-files.ts             # useFileTree / useFile / useUpdateFile
    │   └── use-repos.ts             # useRepos / useBranches / useCheckoutBranch
    │
    ├── router/
    │   └── index.tsx                # React Router 路由定义 + ProtectedRoute / PublicRoute 守卫
    │
    └── pages/                       # 路由页面组件
        ├── login/
        │   └── login-page.tsx       # 登录页（Zod 校验 + React Hook Form）
        ├── register/
        │   └── register-page.tsx    # 注册页（Zod 校验 + 密码确认）
        └── app/
            ├── layout.tsx           # 三栏 IDE 布局外壳（侧栏 + 中栏 Tab + 右栏内容区）
            ├── project-list.tsx     # 左侧栏：项目列表 + 创建对话框
            ├── file-panel.tsx       # 中栏：递归文件树
            ├── file-editor.tsx      # 右栏：文件编辑器（Ctrl+S 快捷键）
            ├── chat-panel.tsx       # 中栏：会话列表 + 删除确认
            ├── chat-view.tsx        # 右栏：SSE 流式 AI 对话 + Markdown 渲染
            ├── repo-panel.tsx       # 中栏：Git 仓库管理（添加/删除/分支切换）
            └── components/          # 页面内子组件
                └── （预留）
```

## 路由设计

| 路径 | 页面 | 认证 | 说明 |
|------|------|------|------|
| `/login` | LoginPage | ✘ | 登录 |
| `/register` | RegisterPage | ✘ | 注册 |
| `/` | AppLayout | ✔ | 重定向到 `/projects` |
| `/projects` | ProjectList | ✔ | 项目列表 |
| `/projects/:projectId/files` | FilePanel | ✔ | 文件树（中栏） |
| `/projects/:projectId/files/*` | FileEditor | ✔ | 文件编辑器（右栏） |
| `/projects/:projectId/chat` | ChatPanel | ✔ | 会话列表（中栏） |
| `/projects/:projectId/chat/:conversationId` | ChatView | ✔ | AI 对话（右栏） |
| `/projects/:projectId/repos` | RepoPanel | ✔ | Git 仓库管理（中栏） |

## 状态管理分工

| 类型 | 工具 | 示例 |
|------|------|------|
| 服务端状态 | TanStack Query | 项目列表、文件内容、会话列表 |
| 全局客户端状态 | Zustand | JWT token、用户信息、选中项目 ID |
| 页面内 UI 状态 | React useState | 中栏 Tab、输入框内容、折叠状态 |
| 表单状态 | React Hook Form | 登录表单、创建项目表单 |
