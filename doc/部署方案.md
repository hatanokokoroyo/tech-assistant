# йғЁзҪІж–№жЎҲ

## Docker Compose жһ¶жһ„

```yaml
services:
  frontend:     # Nginx + React йқҷжҖҒиө„жәҗпјҢеҸҚеҗ‘д»ЈзҗҶ /api вҶ’ backend
  backend:      # FastAPI + AI жЁЎеқ—пјҢеҜ№еӨ– SSE жөҒејҸжҺҘеҸЈ
  postgres:     # PostgreSQL 16
```

## DockerfileпјҡеүҚз«Ҝ

```dockerfile
# frontend/Dockerfile
# Stage 1: build
FROM node:20-alpine AS builder
WORKDIR /app
COPY package.json ./
RUN npm install
COPY . .
RUN npm run build

# Stage 2: serve
FROM nginx:stable-alpine
COPY --from=builder /app/dist /usr/share/nginx/html
COPY nginx.conf /etc/nginx/conf.d/default.conf
EXPOSE 80
CMD ["nginx", "-g", "daemon off;"]
```

```nginx
# frontend/nginx.conf
server {
    listen       80;
    server_name  localhost;

    root   /usr/share/nginx/html;
    index  index.html;

    # еҗҺз«Ҝ API еҸҚеҗ‘д»ЈзҗҶ
    location /api/ {
        proxy_pass http://backend:8000;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;

        # SSE й•ҝиҝһжҺҘж”ҜжҢҒ
        proxy_buffering off;
        proxy_cache off;
        proxy_read_timeout 300s;
    }

    # еүҚз«Ҝи·Ҝз”ұпјҲSPA fallbackпјү
    location / {
        try_files $uri $uri/ /index.html;
    }
}
```

## DockerfileпјҡеҗҺз«Ҝ

```dockerfile
# backend/Dockerfile
FROM python:3.12-slim

WORKDIR /app

RUN apt-get update && apt-get install -y --no-install-recommends \
    git \
    openssh-client \
    && rm -rf /var/lib/apt/lists/*

COPY requirements.txt .
RUN pip install --no-cache-dir -r requirements.txt

COPY . .

EXPOSE 8000

CMD ["uvicorn", "app.main:app", "--host", "0.0.0.0", "--port", "8000"]
```

## docker-compose.yml

```yaml
# docker-compose.yml
version: "3.9"

services:
  frontend:
    build:
      context: ./frontend
      dockerfile: Dockerfile
    ports:
      - "80:80"
    depends_on:
      backend:
        condition: service_started
    networks:
      - app-net

  backend:
    build:
      context: ./backend
      dockerfile: Dockerfile
    env_file:
      - .env
    volumes:
      - tech_assistant_data:/data/tech-assistant
      - ${HOST_SSH_DIR:-./ssh}:/root/.ssh:ro   # еҸҜйҖүпјҡжҢӮиҪҪе…ЁеұҖ git й…ҚзҪ®
    depends_on:
      postgres:
        condition: service_healthy
    networks:
      - app-net

  postgres:
    image: postgres:16-alpine
    env_file:
      - .env
    volumes:
      - pg_data:/var/lib/postgresql/data
    healthcheck:
      test: ["CMD-SHELL", "pg_isready -U ${POSTGRES_USER} -d ${POSTGRES_DB}"]
      interval: 5s
      timeout: 3s
      retries: 5
    networks:
      - app-net

volumes:
  tech_assistant_data:
    driver: local
    driver_opts:
      type: none
      device: ${HOST_DATA_DIR:-./data/tech-assistant}
      o: bind
  pg_data:
    driver: local
    driver_opts:
      type: none
      device: ${HOST_PG_DIR:-./data/postgres}
      o: bind

networks:
  app-net:
    driver: bridge
```

## и®ҫи®ЎиҰҒзӮ№

| еҶізӯ– | зҗҶз”ұ |
|---|---|
| **bind mount** жӣҝд»Ј named volume | ж•°жҚ®еҸҜзӣҙжҺҘеңЁе®ҝдё»жңәжҹҘзңӢе’ҢеӨҮд»ҪпјҢејҖеҸ‘и°ғиҜ•ж–№дҫҝ |
| **postgres healthcheck** | зЎ®дҝқ backend еҗҜеҠЁж—¶ж•°жҚ®еә“е·Іе°ұз»Ә |
| **proxy_buffering off** | SSE жөҒејҸиҫ“еҮәйңҖиҰҒзҰҒз”Ё Nginx зј“еҶІпјҢеҗҰеҲҷеүҚз«Ҝж”¶дёҚеҲ°е®һж—¶жөҒ |
| **proxy_read_timeout 300s** | AI жҺЁзҗҶеҸҜиғҪжҢҒз»ӯж•°еҚҒз§’пјҢйңҖиҰҒй•ҝи¶…ж—¶ |
| **Git е®үиЈ…еңЁ backend е®№еҷЁеҶ…** | AI жү§иЎҢ git е‘Ҫд»Өзҡ„дҫқиө– |
| **backend е®№еҷЁеҶ…ж—  sudo** | еҮҸе°‘е®үе…ЁйЈҺйҷ©пјҢдёҺжІҷз®ұзӯ–з•ҘдёҖиҮҙ |
